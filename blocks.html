Blocks Description

Ceci est une bref description de chaque blocks Scratch inclus avec l'extension scratch Tobby.


LED


Allume (On) ou étteint (Off) toutes les leds

Permet de définir une couleur pour les leds (ex: blanc,rouge...)

Allume toutes les leds avec une couleur précise (ex: blanc= R:255 G:255 B:255)

Allume une led avec une couleur précise

Affiche l'émotion choisi sur la ou les matrices à led


BUZZER


Allume (On) ou étteint (Off) le buzzer

Joue une des mélodie prêt enregistré

Joue la note choisi (do,do#,re,re#,mi,fa,fa#,sol,sol#,la,la#,si)

Joue la note choisi pendant la durée choisi

Configure l'attente du buzzer

Configure le tempo du buzzer

Configure le tempo du buzzer


BOUTON


Lis l'état du bouton selectionné (pressé, relaché)

Passe à l'action suiante quand l'état du bouton selectionné change


MOTEUR


Choisi un moteur (celui de gauche ou de droite ou même les deux), son sens (avant/arrière) et lui associe une vitesse

Arrête un des deux moteurs ou les deux


AUTRE


Attent la connection du robot avant d'éffectuer l'action suivante
