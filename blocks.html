Description des blocs

Ceci est une brève description de chaque bloc Scratch présent dans l'extension Scratch Tobby.


LED


Allume (On) ou éteint (Off) toutes les LED

Permet de définir une couleur pour les LED (ex: blanc, rouge...)

Allume toutes les LED avec une couleur précise (ex: blanc = R:255 G:255 B:255)

Allume une LED avec une couleur précise

Affiche l'émotion choisie sur la ou les matrices à LED


BUZZER


Allume (On) ou éteint (Off) le buzzer

Joue une des mélodies pré-enregistrée

Joue la note choisie (do, do#, re, re#, mi, fa, fa#, sol, sol#, la, la#, si)

Joue la note choisie pendant la durée choisie

Configure l'attente du buzzer

Configure le tempo du buzzer

BOUTON


Lit l'état du bouton selectionné (pressé ou relaché)

Passe à l'action suivante quand le bouton est dans l'état sélectionné


MOTEUR


Choisit un moteur (celui de gauche, de droite ou les deux), son sens (avancer ou reculer) et lui associe une vitesse par rapport à la vitesse maximale possible

Arrête l'un des deux moteurs ou les deux


AUTRE


Attend la connexion du robot avant d'effectuer la ou les actions suivantes
