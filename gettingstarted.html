Manuel d'utilisation

1. Televerser le Firmware dans le robot

  1. Si vous ne l'avez pas déjà, télécharger est installer Arduino (Windows ou Mac instructions)
  2. Connecter le robot Tobby à votre ordinateur grâce au cable USB
  3. Lancer le logiciel Arduino
  4. Ouvrir le projet scratch_programmation.ino
  5. Selectionner l'Arduino MEGA dans Tools > Board menu
  6. Selectionner le port serie sur lequel est connecté le robot Tools > Port menu.
    Sur Mac, c'est quelque chose comme /dev/tty.usbmodem-1511.
    Sur Windows, c'est surement le port COM le plus élevé.
    (Ou unplug Tobby, vérifier le menu, et après replug Tobby et regardez quel port apparaît.)
  7. Clickez sur le button televerser

2. Installer le Plugin Scratch Extensions

Vous devrez utiliser l'explorateur Firefox pour que l'extension marche

Télécharger et installer Plugin Scratch Extensions

3. Charger l'extension de Tobby sur ScratchX

Lancer l'extension en suivant le lien suivant:

http://scratchx.org/?url=http://vannestefelix.github.io/scratch_extensions/lib_global.js

NOTE: Sur Firefox, une fenêtre d'autorisation apparaîtra surement: "Allow scratchx.org to run "Scratch Device?". Click "Allow...".

Un autre message apparaître normalement avec, "Allow scratchx.org to run plugins?". Mettre aussi Adobe Flash que Scratch Device à "Allow and Remember" et clicker sur okay.

4. Attendre le feu vert

Quand vous verrez l'indicateur dans "More Blocks" passer au vert vous pouvez maintenant utiliser l'extension!

Maintenant vous pouvez aller voir Blocks Description and the Example de Projets pour apprendre en detail comment utiliser l'extension.
